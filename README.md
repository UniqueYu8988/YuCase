# 🏥 YuCase - 智能病历解析与自动化填表助手

![YuCase Banner](https://img.shields.io/badge/Status-Stable-success?style=for-the-badge&logo=googleshell&logoColor=white)
![Python](https://img.shields.io/badge/Logic-Python-blue?style=for-the-badge&logo=python&logoColor=white)
![Tkinter](https://img.shields.io/badge/GUI-Tkinter-orange?style=for-the-badge&logo=python&logoColor=white)

**YuCase** 是一套专为医疗文书工作设计的自动化工具包。它通过解析 Word 格式的病历首页文档，自动提取关键信息并构建易于使用的离线数据管理系统，结合全局快捷键实现极速填表。

---

## 🌟 设计初衷

本项目是为母亲开发的专属工具。在日常医疗工作中，繁琐的病案录入耗费了大量精力，YuCase 通过“解析-预览-填报”的三步走方案，旨在将复杂的录入工作转化为简单的“按键”操作。

---

## 🔥 核心功能

1.  **📄 智能解析与提取**：
    - 自动扫描并识别 `.docx` 格式的病历文档。
    - 深度解析患者个人信息（姓名、身份证、地址）、入院时间、诊断结果及医疗费用。
    - **同步策略**：本地文档与离线数据库实时同步，确保数据准确无误。

2.  **🌐 离线可视化数据库**：
    - 基于现代 Web 技术的 `medical_record_lite.html`，提供精美的预览界面。
    - 支持在浏览器中直接修改、校对解析后的数据，并一键同步回本地。

3.  **⌨️ 全局自动化填表助手**：
    - 专为妈妈设计的“大字体、高对比度”桌面客户端。
    - **F9 一键录入**：在任何填报系统光标处，按下 F9 即可自动模拟键入当前字段并跳转至下一项。
    - 支持占位过滤，自动跳过系统中不需要填写的非关键字段。

---

## 🛠️ 技术架构

- **后端解析 (`Python`)**：利用正则匹配与 ZIP 压缩流技术，极速读取 Word 属性。
- **自动化核心 (`win32 API`)**：底层调用 Windows 输入控制，实现毫秒级模拟点击。
- **前端存储 (`Vanilla JS`)**：采用 JSON 持久化方案，数据存储在本地 HTML 中，无需任何数据库服务器，隐私且轻量。

---

## 🔒 隐私声明

**数据安全是本项目的首要准则：**

- **全离线运行**：所有病历数据均保存在本地，不上传至任何云端或服务器。
- **脱敏建议**：建议在使用及上传代码时，定期清理 `病历文档/` 目录及 `medical_record_lite.html` 中的历史记录。

---

## 🚀 极简启动指南

1.  **准备阶段**：将病历 Word 文件放入 `病历文档` 文件夹。
2.  **第一步：更新**：运行 `main_launcher.py` 或打包后的 EXE，点击“更新病历数据”。
3.  **第二步：确认**：点击“查看病历列表”进行最后的核对。
4.  **第三步：填报**：启动“填表助手”，在目标系统光标处按 **F9** 即可。
